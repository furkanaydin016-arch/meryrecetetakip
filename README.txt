Mery Estetik Profesyonel Reçete Takip Sistemi - Web Demo

Kullanım:
1. index.html dosyasını tarayıcıda açın.
2. Demo giriş: admin / 1234
3. Yeni Görüşme bölümünden hasta kaydı oluşturun.
4. Sistem otomatik 2 gün, 7 gün, 30 gün takip görevi oluşturur.
5. WhatsApp butonu hazır mesajla WhatsApp Web açar.
6. Excel / CSV butonu hasta listesini dışa aktarır.

Not:
Bu sürüm profesyonel tasarım demosudur. Gerçek kullanım için:
- Veritabanı
- Gerçek kullanıcı rolleri
- Sunucu kurulumu
- KVKK güvenliği
- Günlük yedekleme
- HTTPS ve erişim logları
eklenmelidir.
